чтоб открыть терминал вводим  в поиск 'терминал'

clear или команд + L чтоб удалить все
чтобы скрыть файл или директорию надо поставить точку перед командой

~ тильда символ домашней директории
/ слеш если одна то это корневая директория (папки)
.. предыдущая директория
. текущая директория 

ls - (лист) команда выводит список всех директорий и файлов в текущей директории  (рабочая директория)
ls -a выводит список скрытых и обычных файлов и директорий 	
ls -l выводит список директорий и файлов с подробнгй информацией 
cd - (change directory) эта команда меняет текущую директорию на указанную 'cd название директории' способ применение)
mkdir - (make directory)  создает указанную директорию 
touch - создает указанный файл 'touch "name of file"' <extension-расширение>
pwd - (print working directory) выводит путь до текущей директории 
rmdir - (remove directory) удаляет указанную директорию (не может удалить не пустую директорию)
rm - удаление файлов 'rm "name of file"'
rm -rf принудительное удаление файла либо директории 
mv - (move) отвечает за перемещение и переименование файла или директории "mv 'файл директория' 'новое имя - директория/'"
cat - выводит содержимое указанного файла 
nano - открывает графический интерфейс в терминале для редактирование файла (если нет файла то создаться новый файл с таким именем) "nano 'name of file"
echo - команда для быстрой записи данных в файл через терминал. "echo 'текст для записи' название файла"
cp - (file to copy) (path) копирует
